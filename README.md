# Nova Studio — Architecture Portfolio

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

**Nova Studio** — Architecture for calm living. A premium portfolio site for an architecture and design studio, featuring project showcases, filtering, and detailed case studies.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Pages

- `/` — Homepage with hero, featured projects, philosophy, services, and contact CTA
- `/projects` — Filterable project gallery (All, Residential, Cultural, Interior)
- `/projects/villa-nova` — Individual project detail with gallery and specifications

## Tech Stack

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript**
- **CSS Custom Properties** (design tokens)
- **Google Fonts** (Inter + Outfit)
